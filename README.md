## 📚 Learning Approach

<div align="center">

| 💡 Principle | 📋 Description |
|:---|:---|
| ✅ **Clean & Readable Code** | Every solution is written to be understood at first glance |
| 🔍 **Logic Over Memorization** | Focus on *why* it works, not just *what* to write |

| 🧱 **Concept-First Structure** | Topics build on each other from ground up |

## ⚡ Getting Started

### Prerequisites

![GCC](https://img.shields.io/badge/GCC_Compiler-A8B9CC?style=flat-square&logo=c&logoColor=black)
![VSCode](https://img.shields.io/badge/VS_Code_(recommended)-007ACC?style=flat-square&logo=visualstudiocode&logoColor=white)
![CodeBlocks](https://img.shields.io/badge/Code::Blocks_(alternative)-f59e0b?style=flat-square&logoColor=white)

```bash
# 📥 Clone the repository
git clone https://github.com/arryansharma1/C-Programming-Practice.git

# 📂 Navigate into the folder
cd C-Programming-Practice

# 🔨 Compile any file using GCC
gcc 01_Basics/variables_datatypes.c -o output

# ▶️ Run the output
./output
```
